Summary

A phishing email with the subject "Free Coffee Voucher" was delivered to user [email protected] (host: 172.16.20.151). The email contained a malicious ZIP attachment that, when executed, deployed Trojan.AsyncRAT/MSIL and established communication with a C2 server in Romania. The host was confirmed compromised and subsequently contained via EDR.

Investigation Walkthrough

1. Initial Alert Review

I began by reviewing all key metadata: event time, SMTP address, sender email, recipient, subject line, and device action. The subject "Free Coffee Voucher" immediately raised suspicion — a free reward theme is a well-known social engineering technique. The device action was marked as Allowed, meaning the email bypassed security controls and reached the user's mailbox.

2. Email & Attachment Analysis

The Email Security section confirmed an attachment named free-coffee.zip. I checked the SMTP address (103.80.134.63) on VirusTotal — returned suspicious. Hybrid Analysis confirmed prior malicious activity on the file hash. Hash verification on VirusTotal confirmed the ZIP as malicious.

3. Payload Analysis

Analysis of ZIP contents revealed an embedded executable coffee.exe, a hardcoded C2 address 37.120.233.226, and malware family identified as Trojan.AsyncRAT/MSIL — a Remote Access Trojan designed to establish a persistent backdoor.

4. Network Log Analysis

Filtering network logs by host IP 172.16.20.151 confirmed outbound connections to 37.120.233.226, confirming active C2 communication. Log review also revealed the download URL used to retrieve the payload.

5. EDR & Endpoint Analysis

EDR telemetry confirmed: network activity to the C2 IP, browser history showing URL access, and coffee.exe was executed. Full host compromise was confirmed across all three data sources.

6. Containment

Deleted the malicious email from the mailbox. Immediately contained the host via EDR to stop C2 communication and prevent lateral movement.

Recommendations
  • Block C2 IP 37.120.233.226 on firewall and EDR
  • Block sender domain coffeeshooop.com in email gateway
  • Remove all malicious files (free-coffee.zip, coffee.exe) from the system
  • Reset credentials of affected user [email protected]
  • Perform full endpoint forensics to identify persistence mechanisms
  • Monitor network logs for lateral movement indicators
  • Conduct phishing awareness training for all users
Key Learnings
  • Analyze the entire attack chain. A phishing attack can escalate from a simple email to full system compromise quickly — focusing on just one indicator misses the full picture.
  • Validate hashes across multiple platforms. Using both VirusTotal and Hybrid Analysis provides stronger confidence in verdicts than a single source alone.
  • Log correlation is essential. No single data source confirmed the compromise — combining email security, EDR, and network logs was critical to building the full picture.
  • Device action "Allowed" means the user is the last defense. When email security passes a malicious email through, user awareness training becomes the critical control.
  • Contain quickly. Recognizing AsyncRAT C2 behaviors and isolating the host fast prevents lateral movement and further damage.
